# 一、单项选择题
1.C
【解析】  $\lim_{x\to 0}\frac{\tan{6}x}{2x} = \lim_{x\to 0}\frac{6x}{2x} = 3$  ，故选C.
2.B
【解析】  $f(x) = \frac{x^2 - x - 6}{x - 3}$  在  $x = 3$  处无定义，且  $\lim_{x\to 3}f(x) = \lim_{x\to 3}\frac{(x - 3)(x + 2)}{x - 3} =$
$\lim_{x \to 3}(x + 2) = 5$ ，故  $x = 3$  是  $f(x)$  的可去间断点，故选 B.
3. D
【解析】根据原函数及不定积分的定义可知，  $F^{\prime}(x) = f(x), \int f(x) dx = F(x) + C$ ，A、B、C项错误，故选D.
4. C
【解析】对于选项A，  $\sum_{n = 1}^{\infty}\frac{1}{n}$  发散，由常项级数的基本性质可知，  $\sum_{n = 1}^{\infty}(u_n + \frac{1}{n})$  发散，对于选项B，因为  $\sum_{n = 1}^{\infty}u_{n}$
收敛，所以  $\lim_{n\to \infty}u_n = 0,\lim_{n\to \infty}(u_n + \frac{1}{2}) = \frac{1}{2}\neq 0,\sum_{n = 1}^{\infty}(u_n + \frac{1}{2})$  不满足级数收敛的必要条件，故发散
对于选项C，  $\sum_{n = 1}^{\infty}\frac{1}{3^n}$  是  $q = \frac{1}{3}$  的等比级数，收敛，又  $\sum_{n = 1}^{\infty}u_n$  收敛，故  $\sum_{n = 1}^{\infty}(u_n + \frac{1}{3^n})$  收敛
对于选项D，  $\sum_{n = 1}^{\infty}\frac{1}{\sqrt{n}}$  是  $p = \frac{1}{2}$  的  $p$  一级数，发散，又  $\sum_{n = 1}^{\infty}u_{n}$  收敛，故  $\sum_{n = 1}^{\infty}(u_n - \frac{1}{\sqrt{n}})$  发散
5. B
【解析】  $\lim_{x\to 0}\frac{f(x)}{g(x)} = \lim_{x\to 0}\frac{\int_{0}^{x^2}\sin t^2dt}{3x^6 + 4x^5} = \lim_{x\to 0}\frac{2x\bullet\sin x^4}{18x^5 + 20x^4} = \lim_{x\to 0}\frac{2x^5}{18x^5 + 20x^4}$
所以当  $x \to 0$  时， $f(x)$  是比  $g(x)$  高阶的无穷小

$$
