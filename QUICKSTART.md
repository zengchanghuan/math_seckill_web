# 快速启动指南

## 🚀 5 分钟快速开始

### 1. 安装依赖

```bash
cd math_seckill_web
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

### 3. 访问应用

打开浏览器访问：<http://localhost:3000>

### 4. 配置后端连接

1. 确保后端服务器正在运行（默认：<http://localhost:8000）>
2. 在网页端进入「设置」页面
3. 配置学生ID（如：`student_001`）
4. 点击「测试连接」确认连接成功

### 5. 开始使用

- **刷题**：进入「刷题」页面，选择主题和难度开始练习
- **推荐**：查看「智能推荐」获取个性化题目
- **画像**：在「学习画像」查看学习数据和分析

## 📝 常见问题

### Q: 无法连接到后端服务器？

A: 检查以下几点：

1. 后端服务器是否正在运行
2. 服务器地址是否正确（默认：<http://localhost:8000）>
3. 是否有跨域问题（CORS）

### Q: 数学公式显示不正常？

A: 确保已安装所有依赖：

```bash
npm install
```

### Q: 如何切换深色模式？

A: 点击导航栏右侧的主题切换按钮，或在「设置」页面切换。
