# 「高数衔接刷题」网页版文案与结构草案

> 目标：  
> 用 **真题 + 基础训练** 做一个简单的网页版，先验证有没有人愿意用、愿意付费。  
> 页面风格：干净、理工风，少图、多信息。

---

## 一、网站整体信息架构（MVP）

- 顶部导航（可选，MVP 可以先不做复杂导航）
  - LOGO / 名字：`高数衔接刷题`
  - 右侧按钮：
    - `登录 / 注册`（可以先用最简单的邮箱/手机号，甚至首版不做）
    - `立即开始刷题`（锚点跳到真题区域）

- 首页（单页即可，往下滚动）
  1. **首屏 Hero 区（卖点 + CTA）**
  2. **功能亮点区（3~4 点）**
  3. **真题列表区（按年份/卷别展示）**
  4. **底部「升级/内测预约」区域**

- 题目练习页
  - 左侧/上方：题目内容
  - 下方：作答区域 + 解析
  - 右侧/顶部：本套进度条、正确率提示

## 三、功能亮点区文案（3~4 个卡片）

可以用三列卡片呈现：

### 卡片 1：真题驱动

- 标题：**从真题出发，而不是泛泛而谈**

  > 精选近几年广东专升本 / 高数相关真题，  
  > 所有题目都按「考点 + 难度」标注，  
  > 刷每一道题都知道：这道题在考什么、值不值得多花时间。

### 卡片 2：重基础、重概念

- 标题：**专门做「高中 → 大学」这一步**
- 文案：
  > 精力不再浪费在极难偏题上，  
  > 而是回到函数、方程、三角、排列组合、复数、参数方程这些  
  > 高数里一定会反复用到的基础模块。

---

### 卡片 3：即时反馈 + 解析

- 标题：**做完就知道为什么对 / 为什么错**

  > 附带「关键思路 + 易错点」解析，  
  > 不再只看到一个冷冰冰的标准答案。

7.1 Pro 版计划（概念草案）

计划中的 Pro 功能包括（仍在设计中）：

自动错题本：所有做错/标记的题自动收集，支持按章节/知识点二刷

专项训练：按「章节 + 难度」刷题，集中攻克某一块

学习进度统计：最近 7/30 天做题数、正确率趋势、薄弱环节分析

真题不断更新：收录更多年份、更多地区的高数真题

预计价格区间：¥19 ~ ¥49，一次付费或短期冲刺卡。

八、小结：首版网页版的目标

不追求「功能完备」，只追求三点：

真题可以正常刷，体验顺畅

基础解析可看，让学生觉得确实有用

你能看见：有没有人愿意用、有没有人愿意付费或留下联系方式
